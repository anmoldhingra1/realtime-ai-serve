# Contributing

Contributions are welcome. Here is how to get started.

## Development Setup

```bash
git clone https://github.com/anmoldhingra1/realtime-ai-serve.git
cd realtime-ai-serve
pip install -e ".[dev]"
```

## Guidelines

- Write tests for new features
- Follow existing code style
- Keep PRs focused: one feature or fix per PR
- Performance-sensitive code should include benchmarks
- Add docstrings to all public methods

## Reporting Issues

Open an issue with a clear description and minimal reproduction steps.
